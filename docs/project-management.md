# Gestión y Organización del Proyecto

## Metodología Aplicada
Se ha implementado la metodología **Kanban** mediante un tablero en Trello. Esto permite visualizar el flujo de trabajo, limitar el trabajo en curso y asegurar un ritmo de desarrollo constante.

## Estructura del Tablero
1. **Backlog:** Ideas y requerimientos funcionales futuros.
2. **Todo:** Tareas técnicas priorizadas listas para ser abordadas.
3. **In Progress:** Tareas en desarrollo actual.
4. **Review:** Código bajo auditoría y pruebas en el dispositivo.
5. **Done:** Funcionalidades integradas, probadas y subidas a GitHub.

Cada funcionalidad se ha descompuesto en subtareas atómicas (checklists) para un control de calidad granular.
