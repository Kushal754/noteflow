# Definición de la Idea: NoteFlow

## 1. Problema que resuelve
La saturación de información obliga a los usuarios a saltar entre diferentes aplicaciones para gestionar textos, tareas o pensamientos rápidos. NoteFlow resuelve esta fricción unificando tres flujos de trabajo en una única interfaz nativa, ligera y privada, que almacena toda la información de forma local.

## 2. Usuario objetivo y Caso de uso
* **Perfil:** Estudiantes, desarrolladores y profesionales.
* **Caso de uso:** Un estudiante utiliza *Notas* para resumir una clase; cambia a *Tareas* para desglosar un proyecto; y usa *Ideas* para registrar un concepto rápido, categorizándolo visualmente con colores y etiquetas.

## 3. Funcionalidades de la Primera Versión (MVP)
* **Navegación:** Separación en tres pestañas nativas e independientes.
* **Notas de Texto:** Creación, lectura y eliminación.
* **Ideas:** Formulario con selector de paleta de colores y agregador dinámico de etiquetas (*tags*).
* **Tareas:** Checklists con capacidad para añadir/eliminar subtareas y guardado inmutable del estado.
* **Persistencia Local:** Almacenamiento con AsyncStorage.

## 4. Funcionalidades Futuras (Backlog Opcional)
* **Búsqueda Global:** Barra de filtrado en tiempo real.
* **Animaciones:** Transiciones con Reanimated.
* **Archivo:** Capacidad para archivar notas históricas.
